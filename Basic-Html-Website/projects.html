Featured Projects

Here are some of the fictional frontend and web development projects I've worked on to practice and showcase my skills.

1. DevTrack – Developer Task Manager

Tech stack: HTML, CSS, JavaScript

A simple yet functional task manager built for developers to track daily goals and productivity. Features include dark mode, drag-and-drop task organization, and local storage persistence.

2. WeatherNow – Weather Forecast Web App

Tech stack: HTML, CSS, JS, OpenWeatherMap API

This project displays current weather and a 5-day forecast for any city. Built with responsive design and API integration. It emphasizes API handling and dynamic DOM updates.

3. QuickCalc – Lightweight Online Calculator

Tech stack: HTML, CSS Grid, Vanilla JS

A basic calculator capable of performing all arithmetic operations. Focused on user experience with keyboard support and clean UI design.

4. PortfoliOne – Single Page Portfolio Template

Tech stack: HTML5, CSS3

A fully responsive portfolio template created using only HTML and CSS. Includes sections for about, projects, skills, and contact. Ideal for beginners looking to build an online presence.

5. CountDownX – Countdown Timer

Tech stack: HTML, CSS, JavaScript

This project lets users set a future date and see a live countdown. It demonstrates DOM manipulation, date/time handling, and user input validation.
